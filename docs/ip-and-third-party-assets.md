## IP Summary
The PeerTutor project primarily consists of **original work** created by the TechTitans team, including the project idea, system design, user flows, wireframes, functional requirements, business model, prototype implementation, testing documents, and project reports.

The project uses several third-party development tools and educational resources, including HTML5, CSS3, JavaScript, Visual Studio Code, Google Sheets, GitHub, and GitHub Classroom. These tools are used under their respective licenses or terms of service.

The team also used **ChatGPT (OpenAI)** as an AI-assisted writing and brainstorming tool to help draft documentation, organize project materials, refine reports, and improve the prototype design. All AI-generated content was reviewed, edited, and verified by the team before inclusion in the final submission.

No proprietary datasets, paid APIs, copyrighted images, commercial fonts, or licensed software components are included in the current MVP. The prototype uses simulated data generated by the project team during usability testing and customer validation. If external icons or visual assets are added in future versions, appropriate attribution and licensing requirements will be followed.
