# Reflection
### Which feature creates the strongest user value?
**F-02 – Tutor Search & Listing** creates the strongest user value because it directly addresses the primary problem identified during customer discovery. Students can quickly search, compare, and find suitable peer tutors based on subject, price, rating, and availability, significantly reducing the time and effort required to locate academic support.

### Which feature creates the strongest business or operational value?
**F-12 – Admin Dashboard & User Management** creates the strongest operational value because it enables administrators to manage users, oversee tutor applications, monitor tutoring activities, and maintain platform quality. Combined with **F-13 (Tutor Application Approval)** and **F-14 (Student Payment Verification)**, it ensures that the platform operates efficiently, securely, and reliably.

### Which required feature is still weak or unclear?
The **Online Payment workflow (F-06, F-14, and F-15)** is currently implemented as a prototype simulation rather than a fully integrated payment system. Future development should incorporate a secure payment gateway, automated payment verification, and real-time payout processing to provide a complete production-ready solution.
