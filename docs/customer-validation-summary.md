# Lab 08 - Customer Validation Summary

## 1. Project Title
**PeerTutor – Student-to-Student Tutoring Marketplace**

## 2. Prototype Tested
The team tested the latest MVP prototype of **PeerTutor**, a web-based platform designed to help Rangsit University students find verified peer tutors through a centralized system. The prototype includes the complete Student, Tutor, and Administrator workflows developed during previous labs.
- **Prototype Version:** MVP Version 1.0
- **Prototype Link or Screenshot Location:** [https://github.com/TSWT-5304/ICT105-TechTitans-MVP/tree/main/wireframes](https://github.com/TSWT-5304/ICT105-TechTitans-MVP/tree/016296eee45b49c2bde6a7c55c8bdbdc272e42b5/wireframes) and `/wireframes/`
- **Main Tasks Tested:**
  - Student registration/login
  - Tutor search and filtering
  - Tutor profile viewing
  - Tutoring request submission
  - Request status tracking
  - Simulated online payment
  - Tutor registration and profile management
  - Tutor dashboard request management
  - Admin tutor approval
  - Payment verification
  - Dashboard analytics

## 4. Validation Method
The MVP prototype was evaluated through task-based usability testing. Each participant completed tasks according to their assigned role (Student, Student Tutor, or Administrator) while the team observed task completion, navigation, and user interactions. After completing the tasks, participants completed a feedback questionnaire and provided suggestions for improvement.
- **Testing Method:** Task-based usability testing with a clickable web prototype
- **Date/Time:** July 16, 2026
- **Location or Online Platform:** Computer Laboratory / Online Demonstration (Google Meet)
- **Number of Testers:** 10
- **Data Collected:**
  - Task completion results
  - Completion time
  - Observation notes
  - User feedback questionnaire
  - Overall usability rating
  - Improvement suggestions
