## Project Information
**Project Title:** PeerTutor – Student-to-Student Tutoring Marketplace

**Group Name:** TechTitans

**Prototype Platform:** Frontend Web Application (HTML, CSS, JavaScript) with Google Sheets

# 1. Overview
The purpose of this analytics report is to analyze the customer validation results collected during the PeerTutor MVP usability testing. The analysis evaluates user task completion, satisfaction, usability, and requirement validation to determine whether the MVP successfully addresses the needs identified during earlier customer discovery activities.

The evaluation is based on feedback collected from 10 participants representing the three primary user groups: Students, Student Tutors, and Administrators.

# 5. Feature Performance
## Student Features
The Student workflow achieved the highest usability score. Participants found tutor searching, tutor profile viewing, tutoring request submission, and payment simulation straightforward.

The search and filtering functions significantly reduced the effort required to compare tutors compared with traditional Facebook and LINE groups.

## Student Tutor Features
Tutor registration, tutor application submission, and tutor profile management were completed successfully.

Participants appreciated the structured profile management system and Tutor Earnings section which are easier to understand.



## Administrator Features
Administrator participants successfully completed tutor approval, payment verification, user management, and dashboard analytics.

Dashboard summary cards effectively supported platform monitoring and administrative decision-making.

# 6. Usability Analysis
The majority of participants described the interface as clean, organised, and easy to navigate.
Several minor usability issues were identified.
| Issue | Frequency | Priority |
|------|----------:|---------|
| Payment status wording | Medium | Medium |
| Confirmation message visibility | Low | Low |
| Dashboard navigation labels | Low | Low |

None of these issues prevented users from completing the assigned tasks.

# 7. User Satisfaction
Participants expressed positive opinions regarding the overall prototype.
Positive feedback included:
- Easy navigation.
- Clear tutor information.
- Fast tutor searching.
- Useful dashboard summaries.
- Well-organised workflow.

Users particularly appreciated that PeerTutor centralizes tutor discovery into a single platform instead of relying on Facebook groups, LINE groups, or personal recommendations.

# 8. Key Insights
The customer validation provides several important insights.

### Strengths
- Centralized tutor discovery addresses the identified user problem.
- Search and filtering improve tutor selection.
- Tutor profiles support informed decision-making.
- Dashboard summaries improve task monitoring.
- Administrator workflows are easy to understand.
- Responsive interface supports different devices.

### Opportunities for Improvement
- Improve payment status descriptions.
- Add notification badges for new requests.
- Improve dashboard navigation.
- Increase visibility of confirmation messages.

These improvements focus on usability enhancements rather than functional corrections.

# 9. MVP Evaluation
Based on the validation results, the PeerTutor MVP successfully demonstrates the primary workflows required by Students, Student Tutors, and Administrators.

The prototype achieved:
- High task completion
- Positive user satisfaction
- Complete Functional Requirement coverage
- No critical usability failures

The identified issues are minor interface improvements that can be implemented without changing the overall system design.

# 10. Recommendation
The validation findings indicate that the PeerTutor MVP is ready to progress to the next stage of development.

The team recommends:
- Continue developing the current MVP.
- Improve user interface clarity.
- Enhance notification and feedback messages.
- Refine dashboard navigation.
- Prepare backend integration for future versions.

Overall, the customer validation confirms that PeerTutor effectively addresses the original customer problem and provides a practical, user-friendly tutoring marketplace for students, tutors, and administrators.
